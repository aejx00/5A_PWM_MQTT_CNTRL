# 5A PWM CNTRL MQTT

PWM Control via MQTT


## Description

Using a Node MCU dynamically control a 5 Amp PWM Controller via the MQTT protocol


### Dependencies
Based on Robojax Design:
https://www.youtube.com/watch?v=en-SJookvjo

### Installing
* Clone project
* Update config
* Flash Node MCU


### Configuration

* Update MQTT Details
* Update WIFI Details
